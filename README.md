# 🚀 Minecraft Launcher (Beta)

A lightweight, Python-based Minecraft launcher designed for speed and simplicity. 

> **Note:** This project is currently in the **early beta stage**. The UI is functional but minimalist, as I am focusing on core features and stability first.

### ✨ Features
* **Standalone executable:** No need to install Python or extra libraries.
* **Plug & Play:** Just run the file and you're ready to go.
* **Lightweight:** Minimal system impact.

### 📥 How to use
1. Go to the **[Releases](https://github.com/sweatz-arch/minecraft-launcher-beta/releases)** section on the right.
2. Download the latest `.exe` file.
3. Run the executable and enjoy!

### ⚠️ Development Status (WIP)
* **GUI:** Currently under development (Basic/Minimalist style).
* **Testing:** Expect some bugs, as this is a test phase.
* **Updates:** Regular improvements coming soon!
